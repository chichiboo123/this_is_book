# 📚 '책임(This is Book)' - 초등학생 독서 습관 웹앱 제작 프롬프트

## 앱 개요
초등학생의 독서 습관을 길러주기 위한 웹앱 **'책임(This is Book)'**을 만들어줘.

---

## 1. 책 검색 (Google Books API)
- Google Books API를 연동하여 **책 제목 또는 저자명**으로 검색
- 검색 결과에서 책을 선택하면 다음 정보를 표시: **제목, 저자, 출판일, 페이지 수, ISBN, 언어, 표지 이미지, 주요 내용(description)**
- 검색 결과는 리스트로 보여주고, 클릭하면 상세 정보가 펼쳐짐

## 2. 책 카드 만들기 (캐릭터 의인화)
- 책을 의인화하여 캐릭터로 발전시키는 활동
- 입력 필드: **캐릭터 이름, 취향, 외모, 자주 하는 말, 어울리는 색깔**
- 책에 어울리는 **이모지 선택** 기능
- **이미지 업로드** 또는 **앱 내 드로잉 도구**(HTML5 Canvas 기반)로 직접 그림을 그려 캐릭터화
- 드로잉 도구: 펜 색상 선택, 굵기 조절, 지우개, 전체 지우기 기능
- 입력한 내용이 실시간으로 **프리뷰 카드**에 반영됨

## 3. 소개 카드 만들기
- 이 책에 대한 **기대평** 또는 **홍보 문구**를 작성하는 카드
- 텍스트 입력 영역 제공
- 실시간 프리뷰 카드 반영

## 4. 질문 카드 만들기
- 책을 보고 떠오르는 **질문을 기록**하고 **스스로 답도 작성**
- 질문/답변 쌍을 **동적으로 추가/삭제** 가능한 리스트 형태
- 실시간 프리뷰 카드 반영

## 5. 내보내기 기능
- 3개 카드(책 카드, 소개 카드, 질문 카드) 각각 개별 내보내기 가능
- **체크박스로 선택**하여 여러 카드를 동시에 내보내기 가능
- 내보내기 형식: **WebP 이미지 다운로드**, **클립보드 복사**
- `html-to-image` 라이브러리 사용

## 6. 테마 (파스텔 4종)
- 초등학생에게 어울리는 귀여운 디자인
- **파스텔 블루, 파스텔 그린, 파스텔 옐로우, 파스텔 핑크** 4가지 테마
- 헤더에 하트 이모지 버튼으로 테마 선택 시 전체 앱 색상이 자동 변경
- CSS 변수(`--primary`, `--background`, `--card` 등)로 테마 시스템 구현

## 7. 다국어 지원 (4개국어)
- **한국어(기본), 영어, 일본어, 중국어** 지원
- 언어 토글 버튼으로 전환
- 모든 UI 텍스트가 선택된 언어로 표시됨
- `i18n.ts` 파일에서 번역 관리

## 8. 레이아웃
- 각 섹션(책 검색, 책 카드, 소개 카드, 질문 카드)은 **2컬럼 그리드**(데스크톱): 왼쪽 입력 활동 영역 + 오른쪽 프리뷰 카드
- 모바일에서는 1컬럼으로 반응형 전환
- 상단 헤더: 앱 타이틀 + 테마 선택 + 언어 토글 (sticky)

## 9. 폰트
- 기본 폰트: **Pretendard** (`@import url("https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.min.css")`)
- 타이틀/포인트 폰트: **Black Han Sans** (Google Fonts)
